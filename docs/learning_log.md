# 3 失败案例记录

记录系统错误非常重要。

---

## 实验编号

```
EXP-002
```

---

## 问题类型

```
误识别字段说明表
```

---

## 输入文档

```
字段名 | 类型 | 说明
host_ip | string | 资产IP地址
```

---

## 模型输出

```json
{
 "field": "host_ip",
 "type": "enum",
 "mapping": {
  "string": "资产IP地址"
 }
}
```

---

## 问题分析

模型误将 **字段说明表** 识别为枚举。

原因：

```
表格结构类似 key-value
```

---

## 解决方案

新增规则：

```
字段说明表不识别为枚举
```

更新：

```
extraction_spec.md
```
